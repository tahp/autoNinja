Romulator PROM Files Readme
===========================

This folder contains the PROM files for the FPGA located on the Romulator board. Here is the detail of those files:

* USB_romu_v1_2.iic

* Romulator_Jade_20050221.pof
This PROM must be used when using the Romulator with a Jade based device.

* Romulator7_2G_8v3a.pof
This PROM must be used when using the Romulator version 7 (not with version 6) with a Modem2G (Granite, Greenstone, Nephrite, Emerald...) based device.
